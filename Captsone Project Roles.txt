This is a brief idea of what each student can do; you can be flexible with this, but in the end it would be good 

Student 1 — Simulator Setup & Environment

Set up AirSim/SITL and get the drone working in simulation.

Student 2 — Data Processing & Airsim Environment Development

Extract weed zones / coordinates (using the sim4_navigtion_py file as an example). Also build a plant/weed filled field in Airsim with Unreal Engine.  

Student 3 — Spray Decision Algorithm

Develop logic to classify areas into spot spray vs blanket spray.

Student 4 — Integration & Mission Execution

Connect the decision output to navigation or simulated spraying in the simulator.
